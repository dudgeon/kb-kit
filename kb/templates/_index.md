# Entity types

One file per type; the file both defines the type and serves as its template.
Types are grown, not designed: add one only when ingestion keeps producing
pages that don't fit. Adding a type = add a file here + note it in
[_log.md](../_log.md).

- [note](./note.md) - default type for a distilled piece of knowledge.
- [source](./source.md) - an external artifact this KB has ingested.
- [decision](./decision.md) - a choice that was made, with rationale.
- [topic](./topic.md) - a concept page synthesizing multiple sources.
- [standard](./standard.md) - a published spec or convention (demo type).
- [tool](./tool.md) - a product/system in the landscape (demo type).
