# Tools & systems

Products and systems in the landscape; each page links the idea it embodies
and the standards it implements.

- [OpenWiki](./openwiki.md) - LangChain's agent-maintained wiki CLI; first major OKF producer.
- [Letta](./letta.md) - MemGPT lineage; Feb 2026 pivot to git-based context repositories.
- [Mem0 / OpenMemory](./mem0.md) - memory-as-a-service; one memory ported across agent stacks.
- [Obsidian](./obsidian.md) - the vault-as-agent-KB pattern; human UX over agent-native markdown.
- [ChatGPT memory](./chatgpt-memory.md) - the largest deployed memory system; opaque, non-portable outlier.
- [brainkit](./brainkit.md) - forkable second-brain kit; the maintainer's public prior work.
- [duo](./duo.md) - macOS pairing workspace defining the OKF-aligned vault flavor this kit borrows from.
