# Letta

Letta is the company behind MemGPT, the OS-inspired approach to agent memory:
the agent manages its own memory the way an operating system manages virtual
memory, paging information in and out of context and editing its own memory
blocks.

The event that earns Letta a page in this survey is its **February 2026
rebuild around git-based "context repositories."** The flagship memory company
moving its substrate to versioned files is the strongest single signal that
agent memory and knowledge bases are merging — the convergence traced in
[agent memory](../topics/agent-memory.md) — and it lands Letta squarely on the
[context repo](../topics/context-repo.md) thesis: memory as a durable,
inspectable, diffable asset rather than opaque vendor state (contrast
[ChatGPT memory](./chatgpt-memory.md)).

Credibility marker: Letta Code was the #1 model-agnostic open-source agent on
Terminal-Bench as of 2026-06.

Relationship to standards: the research notes do not record Letta emitting
[OKF](../standards/open-knowledge-format.md) bundles — its context
repositories are its own format on the shared markdown-in-git substrate.
Whether the memory vendors and the knowledge-format camp converge on one
bundle format is an open question worth tracking alongside
[Mem0](./mem0.md)'s portability work.
